Problemas do estilo inline

Um dos principais problemas do estilo inline é que o seu código fonte fica totalmente poluido e confuso no decorrer do desenvolvimento

Apesar de ser uma sintaxe valida pra HTML5 e CSS3 o estilo inline deixa o código extremamente confuso.
